Basic Text Formatting Tags In HTML

Tag Use Description
<b></b> Bold Text Makes the text on which it is applied bold and dense.It is considered an obsolete tag and has been replaced by the strong tag
<strong></strong> Strong Importance Tag It is used to refer to elements of greater importance, visually it is equivalent to the bold tag, but it brings a semantic difference during parsing by the browser,it supercedes the bold tag
<i></i> Italicize Text Makes the text on which it is applied italisized.It is considered obsolete and has been replaced by the em tag
<em></em> Emphasize Text Makes the text on which it is applied emphasized that is italisized.
<small></small> Smaller Text Makes the text on which it is applied smaller relative to the other text.
<mark></mark> Mark Text Makes the text on which it is applied highlighted(with yellow color by default).
<del></del> Deleted Text Strikethrough text
<ins></ins> Inserted/Underline Text Refers to text inserted into the document, underlines text
<sub></sub> WriteSubscript Text Writes text as subscript
<sup></sup> WriteSuperscript Text Writes text as superscript
