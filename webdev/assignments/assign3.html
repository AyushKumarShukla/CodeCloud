Operating System Families

  1. UNIX FAMILY

      Description:These operating systems are based on the idea of the UNIX operating system also called the "UNIX Philosophy" in which several small utilities that serve defined purposes are used in collection to bring about a process. They also include different shells that allow users to communicate with the kernel and get services performed based upon a set of commands input in the command line.

      Some Examples:
    • LINUX DISTRIBUTIONS/LINUX BASED
      • UBUNTU
      • KALI LINUX
      • FEDORA
      • ARCH LINUX
      • ANDROID
      • CHROME OS
    • BSD
    • OPENSOLARIS
  2. DOS FAMILY

      Description:MS-DOS was also a command line based operating system that was built to work on preliminary microprocessing systems. It provided compatibility with a wide range of hardware systems.

      Some Examples:
    • EARLY VERSIONS OF MICROSOFT WINDOWS
