Pointers Prowess

Computer memory can store instruction and data, but the idea of a pointer is revolutionary, a pointer is defined as a memory location that contains a binary pattern which is the address of some other memory location, this might seem like a confusing thing at first, but once we start delving deeper and use pointers we will realise that they are inseparable from the world of computer programming, the very architecture of the microprocessors that we use have pointer manipulation built into them.
