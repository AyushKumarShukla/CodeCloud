The Magic Of Recursion

Recursion is one of the most mind boggling concepts in computer science,which seems to magically solve problems, it works on a very simple process, but the semantics of it, and its execution quickly escalate to astonishing levels of complexity.

Mathematically, a recursive function is a function which is defined in terms of itself, that is the values generated by the function depend on a set of previous values generated by it in the past.A classic example of this is the factorial if we wish to find the factorial of 5 ie 5!, we can simply multiply 5 with the value of 4!, but here, as we can see, we encounter another factoral function which demands us to know the value of 4!, proceeding as we did before, the following list represents the semantics of this process

  • 5! = 5*4!
  • 4! = 4*3!
  • 3! = 3*2!
  • 2! = 2*1!
  • 1! = 1*0!
  • 0! = 1 (this is the only value that we know absolutely(base case))
  1. 0! = 1
  2. 1! = 1*0! = 1*1 = 1
  3. 2! = 2*1! = 2*1 = 2
  4. 3! = 3*2! = 3*2 = 6
  5. 4! = 4*3! = 4*6 = 24
  6. 5! = 5*4! = 5*24 = 120
